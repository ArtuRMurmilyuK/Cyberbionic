Task
Using Visual Studio, create a project using the Console Application template.
Required:
Create a Converter class.
In the body of the class, create a custom constructor that takes three real arguments,
and initializes the fields corresponding to the rate of 3 major currencies, in relation to the hryvnia - public
Converter (double usd, double eur, double rub).
Write a program that will convert from hryvnia to one of the specified currencies,
the program should also convert from the specified currencies to the hryvnia.

Task
Create a class that represents the classroom ClassRoom. 
Create a class Pupil student. 
In the body of the class, 
create methods void Study (), 
void Read (), 
void Write (), 
void Relax (). 
Create 3 derived classes ExcelentPupil, GoodPupil, BadPupil 
from the base class Pupil and override each of the methods based on the student's progress. 
The constructor of the ClassRoom class takes arguments of type Pupil, and the class must have 4 students. 
Allow for the possibility that the user can pass 2 or 3 arguments. 
Display information about how all the students of an instance of the ClassRoom class can learn, read, write, and relax.

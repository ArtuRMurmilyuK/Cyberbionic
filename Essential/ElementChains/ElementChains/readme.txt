 - Task
  - Using Visual Studio, create a project using the Console Application template.
  - Create a program that will display strings of falling characters. 
  - The length of each chain is randomly assigned. 
  - The first character in the chain is white, the second character is light green, 
  - the rest of the characters are dark green. 
  - During the fall of the chain, at each step, all the symbols change their meaning. 
  - Having reached the end, the chain disappears and a new chain is formed on top.
